What it does

Everything you need, nothing they can see

Built for situations where you need AI help but can't afford to trip detection systems.

🕵️

Invisible to detection

Spoofs visibilityState, document.hidden, hasFocus(), and fullscreenElement. Silently swallows blur, visibilitychange, and focusout events before they fire.

💬

AI chat overlay

Ask anything via a floating overlay. Supports full conversation history, markdown rendering with code blocks, and multi-turn dialogue — routed through the Anthropic API using your own key.

📸

Screenshot + vision

Hit Alt+Shift+Z to capture exactly what's on screen and get an AI response. The overlay hides itself during capture so the screenshot is clean, then reappears with the answer.

🔒

No data leaves your machine

Conversations go from the extension to your local Go server, then directly to the Anthropic API with your own key. No third-party relay, no logging, no accounts.

🧲

Draggable overlay

The chat panel floats above everything at maximum z-index. Drag it anywhere, minimize it to a title bar, or dismiss it instantly. It remembers your position across the session.

🦊

Chrome & Firefox

Two dedicated builds from the same codebase. The Chrome build runs full stealth injection in the main world. The Firefox build loads the overlay and vision mode without the spoof layer.

How it works

Simple by design

A TypeScript extension talking to a Go backend. The backend is already hosted on Render — self-hosting is possible but requires tweaking the server URL in the source.

1

The backend is already running

The Go server is hosted on Render and configured by default in the extension — no setup needed to get started. To self-host, clone the repo, point the extension at your own server URL, and supply your ANTHROPIC_API_KEY.

2

Load the extension

Build with npm run build, then load the dist/ folder as an unpacked extension in Chrome, or load the manifest file in Firefox. Takes about 30 seconds.

3

Open any tab and press Ctrl+Shift+X

The chat overlay appears. The inject script has already patched the page's focus and visibility APIs at document_start — the page sees you as fully focused the entire time.

4

Ask or screenshot

Type a question and hit Enter, or press Alt+Shift+Z for vision mode. Responses come back from the AI. Conversation history is preserved until you click Clear or close the browser.

Alt+Shift+Z Screenshot + ask AI

Get started

Free & open source

The backend runs on Render out of the box. To self-host, you'll need Node.js, Go, and your own Anthropic API key — plus a small source code change to point the extension at your server.

Chrome

Full stealth mode · MV3
  1. Clone the repo and run npm install in extension/
  2. Run npm run build to generate extension/dist/
  3. Go to chrome://extensions, enable Developer Mode
  4. Click Load unpacked and select the dist/ folder
  5. Start the Go server and open any tab
Clone on GitHub

Firefox

Overlay + vision · MV3
  1. Clone the repo and run npm install in extension/
  2. Run npm run build:firefox to generate the Firefox build
  3. Go to about:debugging → This Firefox
  4. Click Load Temporary Add-on… and select dist/manifest.json
  5. Start the Go server and open any tab
Clone on GitHub
Self-hosting: Requires Node.js ≥ 18 · Go ≥ 1.21 · An Anthropic API key. Add the key to server/.env, then update the server URL in the extension source to point at your own instance before building.
