What it does

Everything you need,
nothing they can see

Built for situations where you need AI help but can't afford to trip detection systems.

Invisible to detection

Spoofs visibilityState, document.hidden, hasFocus(), and fullscreenElement. Silently swallows blur and focusout events before they fire.

AI chat overlay

Ask anything via a floating panel. Full conversation history, markdown with code blocks, multi-turn dialogue — routed through the Anthropic API with your own key.

Screenshot + vision

Capture exactly what's on screen and get an AI response. The overlay hides itself during capture for a clean shot, then reappears with the answer.

Your key, your data

Conversations go from the extension to the backend, then directly to Anthropic with your API key. No relay, no logging, no third-party accounts.

Draggable overlay

The chat panel floats at maximum z-index. Drag it anywhere, collapse it to a title bar, or dismiss it instantly. Position persists across the session.

Chrome & Firefox

Two builds from one codebase. Chrome gets full stealth injection in the main world. Firefox gets the overlay and vision mode without the spoof layer.

How it works

Simple by design

A TypeScript extension talking to a Go backend hosted on Render. Self-hosting is possible but requires tweaking the server URL in the source.

1

The backend is already running

The Go server is hosted on Render and set as the default in the extension — no server setup needed to get started. To self-host, clone the repo, point the extension at your own URL, and supply your ANTHROPIC_API_KEY.

2

Load the extension

Build with npm run build and load the dist/ folder as an unpacked extension in Chrome, or use npm run build:firefox and load the manifest file in Firefox. Takes about 30 seconds.

3

Open any tab and press Ctrl+Shift+X

The overlay appears. The inject script has already patched the page's focus and visibility APIs at document_start — the page sees you as fully focused the entire time.

4

Ask or screenshot

Type a question and hit Enter, or press Alt+Shift+Z for vision mode. Responses come back from the AI. History is preserved until you click Clear or close the browser.

Chrome

Full stealth mode · MV3
  1. Download and unzip the extension package below
  2. Go to chrome://extensions, enable Developer Mode
  3. Click Load unpacked and select the unzipped folder
  4. Open any tab and press Ctrl+Shift+X

Firefox

Overlay + vision · MV3
  1. Download and unzip the extension package below
  2. Go to about:debugging → This Firefox
  3. Click Load Temporary Add-on… and select manifest.json from the unzipped folder
  4. Open any tab and press Ctrl+Shift+X
Download for Firefox
Self-hosting: Requires Node.js ≥ 18 · Go ≥ 1.21 · An Anthropic API key. Add the key to server/.env, then update the server URL in the extension source before building.
